# 问题3：SMT 贴片机拾取-放置路径优化 —— 严谨数学模型

---

## 一、符号说明

| 符号 | 含义 | 单位 |
|------|------|------|
| \(M = \{1, 2, \ldots, 100\}\) | 贴装点集合（100 个） | — |
| \(F = \{1, 2, \ldots, 30\}\) | 送料器槽位集合（30 个） | — |
| \((X^{\text{mount}}_i, Y^{\text{mount}}_i)\) | 第 \(i\) 个贴装点的坐标 | mm |
| \((X^{\text{feeder}}_f, 0)\) | 第 \(f\) 个槽位的坐标，\(X^{\text{feeder}}_f = 5 + 10(f-1)\) | mm |

| \(\pi\) | 拾取-放置的操作序列 | — |
| \(Q\) | 吸嘴容量（子问题 1：\(Q=1\)；子问题 2：\(Q=2\)） | 个 |
| \(d_{\text{M}}\) | 曼哈顿距离：\(d_{\text{M}}(p,q) = \vert p_x-q_x\vert + \vert p_y-q_y\vert\) | mm |

## 二、模型假设与论证

**假设 1：距离度量为曼哈顿距离。**

论证：贴片机的 X 轴和 Y 轴由独立电机驱动，运动为两轴位移的叠加。在时间最优控制下，X 和 Y 同时启停，路径为折线，实际移动距离即曼哈顿距离。这与题目说明"设备 X/Y 轴独立运动"一致。

**假设 2：同槽位的元件按顺序取用。**

论证：送料器是线性排列的带状供料装置，元件在料带中依次排列。编号在前的元件必须先被取用，这是物理顺序约束。

**假设 3：元件与槽位一一对应。**

论证：本题中每个槽位供应一种特定类型的元件。不同槽位之间元件类型不重复。因此"将元件分配至槽位"等价于"将贴装点按元件类型分组至对应槽位"——但题目原始数据中的 Feeder_ID 是随机初始分配，可被优化。

**假设 4：吸嘴拾取和放置时间忽略不计。**

论证：拾取和放置动作的机械时间在毫秒量级，而空移时间在秒量级（移动距离 100-300 mm，速度约数百 mm/s），相对差距在两个数量级以上。优化空移路径是主要矛盾。

**关于起始位置：** 吸嘴初始位置设为 PCB 板面中心区域 (150, 100) mm，对应于贴片机底座的自然待机位置。该位置对整体路径的影响可忽略（仅影响第一步的到 feeder 距离），且与贴装点分布的中心区域接近。

**假设 5（子问题 3(2)）：预拾取模式下两个元件必须来自同一槽位。**

论证：吸嘴在拾取第二个元件前不移动，因此两个元件只能从同一槽位连续拾取。这是物理约束。

**假设 5'（子问题 3(3) 双向取料的修正）：** 当吸嘴可从两个相邻送料器槽位各取一个元件时，配对范围扩展至槽位对 \((2k-1, 2k)\)。相邻槽位间距仅 10 mm，拾取两个元件后贴装至各自目标位置，可消灭所有单件任务。该假设来自彭乾伟[10]的双向取料策略，是假设 5 在跨槽维度上的推广。

## 三、子问题 3(1)：单件取放（Q=1）

### 3.1 问题结构

子问题 3(1) 包含**两个嵌套决策层**：

- **槽位分配**：100 个贴装点 → 30 个槽位，满足均衡约束（每槽 3-4 个）
- **操作序列**：100 次交替的 P→M 循环（P = 拾取，M = 贴装）

### 3.2 槽位分配优化

**目标：** 最小化贴装点与其分配槽位之间的 X 方向距离期望值。

**定理 2（X-排序分配的最优性）：** 将所有贴装点按 X 坐标升序排列，将所有槽位按 X 坐标升序排列，依次分配（前 10 槽各 4 个，后 20 槽各 3 个），可在满足均衡约束下极小化 \(\sum_i \vert X^{\text{mount}}_i - X^{\text{feeder}}_{a(i)}\vert\)。

**证明思路：** 这是一个带容量约束的一维匹配问题。门格尔（Monge）性质保证排序后顺序匹配是最优的。均衡约束通过控制每槽容量（前 10 槽容量 4，后 20 槽容量 3）实现，恰好满足 100 个点的分配。\(\square\)

**实验验证：**

| 分配方式 | 平均 \(\vert\Delta X\vert\) (mm) | 标准差 |
|----------|:---:|:---:|
| 数据原始（随机） | 89.52 | 54.3 |
| X-排序优化 | **17.38** | 10.2 |
| **改进幅度** | **↓ 80.6%** | **↓ 81.2%** |

**注：** 定理 2 仅针对槽位分配子问题——即最小化 feeder→mount 的 X 分量之和。贴装点之间的 mount→mount 空移、Y 方向距离（mount_y − 0）与 feeder 分配无关，需通过后续的贪心调度额外优化。完整的路径优化包含"槽位分配 + 操作序列调度"两个阶段。

### 3.3 操作序列优化

给定槽位分配 \(a_{i,f}\) 后，每个槽位 \(f\) 有一组待贴装的点 \(M_f = \{i : a_{i,f} = 1\}\)。操作序列 \(\pi\) 需交替访问槽位和贴装点。

**贪心调度算法：**

```
Algorithm: Greedy_Single_Pick_Place(M_f for f∈F)
  Input:  每个槽位的贴装点集合（已按距槽位距离升序排列）
  Output: 操作序列 π, 总曼哈顿距离 Z₁
  
  current_pos ← (150, 100)  // 从 PCB 中心区域起始
  remaining  ← {(f, i) : i ∈ M_f, f ∈ F}  // 所有待处理的 (槽位, 贴装点) 对
  Z₁ ← 0
  
  while remaining ≠ Ø:
    // 选使下一步总距离最小的 (槽位, 贴装点)
    (f*, i*) ← argmin_{(f,i) ∈ remaining} [ dM(current_pos, feeder_f)
                                            + dM(feeder_f, mount_i) ]
    Z₁ ← Z₁ + dM(current_pos, feeder_f*) + dM(feeder_f*, mount_i*)
    current_pos ← mount_i*
    π.append((f*, i*))
    remaining.remove((f*, i*))
  
  return π, Z₁
```

## 四、子问题 3(2)：预拾取（Q=2）

### 4.1 配对决策

同槽位内的贴装点需两两配对（或余下单件处理）。配对的目标是使成对的两个贴装点尽可能靠近，从而减少两次贴装之间的空移。

**配对策略对比：**

| 策略 | 描述 | 总距离 (mm) |
|------|------|:---:|
| **A — 最近邻配对** | 同槽位内，每次选曼哈顿距离最近的两个点配对 | **23,688** |
| B — X 坐标相邻配对 | 同槽位内，按 X 坐标排序后相邻配对 | 24,758 |
| C — 最远优先配对 | 同槽位内，每次选距离最远的两个点配对 | 26,954 |

策略 A 最优，后续分析基于策略 A。

### 4.2 路径距离计算

**单对操作（P→P→Mₐ→M_b）：**

两个贴装点的访问顺序影响总距离。需比较：

\[
d_1 = d_M(\text{prev}, F) + d_M(F, M_a) + d_M(M_a, M_b)
\]
\[
d_2 = d_M(\text{prev}, F) + d_M(F, M_b) + d_M(M_b, M_a)
\]

取 \(\min(d_1, d_2)\) 并记录相应的访问顺序。

**单件操作（P→M）：**

\[
d = d_M(\text{prev}, F) + d_M(F, M)
\]

### 4.3 贪心调度

```
Algorithm: Greedy_Double_Pick_Place(tasks)
  Input:  tasks = {(f, type, mount1, mount2)}  // type ∈ {pair, single}
  Output: 总曼哈顿距离 Z₂
  
  current_pos ← (150, 100)
  remaining ← all tasks
  Z₂ ← 0
  
  while remaining ≠ Ø:
    // 选下一步总距离最小的任务
    t* ← argmin_{t ∈ remaining} PathCost(t, current_pos)
    Z₂ ← Z₂ + PathCost(t*, current_pos)
    current_pos ← LastPosition(t*)
    remaining.remove(t*)
  
  return Z₂
```

## 五、子问题对比分析

| 指标 | 单件取放 (Q=1) | 预拾取 (Q=2) | 改善 |
|------|:---:|:---:|:---:|
| 总曼哈顿距离 | 36,324 mm | 23,688 mm | **↓ 34.8%** |
| 任务数 | 100 | 60 (40 对 + 20 单件) | ↓ 40 次往返 |
| 到槽位的往返次数 | 100 次 | 60 次 | ↓ 40% |

**改善机理：** 预拾取将容量从 1 提升至 2，使吸嘴每次到槽位可"批量"处理两个元件。每对配对节省一次从贴装点返回槽位的空移。对于 40 对配对，共节省了 40 次"贴装点→槽位"的往返。

**边际效益分析：** 进一步提升至 Q=3 的边际收益递减——已配对 80 个元件（40 对），仅余 20 个单件。Q=3 可再减少约 10 次往返，但喷嘴重量约束和机械复杂性增加。Q=2 是工程上的合理折中。

## 六、子问题 3(3)：双向取料（跨槽位预拾取）

子问题 3(2) 的预拾取策略要求配对的两个元件来自同一槽位，导致奇数容量槽位必然产生单件任务（共 20 个单件）。受彭乾伟[10]的 SMT 双向取料策略启发，本子问题将配对范围扩展至**相邻槽位对**：吸嘴在相邻两个 feeder（如 Feeder 1 和 Feeder 2，间距仅 10 mm）各取一个元件后，一次贴装两个元件。

### 6.1 送料器配对与跨槽配对

将 30 个送料器按物理相邻关系配为 15 对 \((2k-1, 2k)_{k=1}^{15}\)：

- 前 5 对（槽位 1–10）各含 8 个元件（4+4）
- 后 10 对（槽位 11–30）各含 6 个元件（3+3）

所有对均为偶数容量 → **跨槽配对可消耗所有元件，消灭全部单件任务**。

每对内的元件通过贪心最近邻跨槽配对：每次选取两个送料器中曼哈顿距离最近的一对未配对元件。

### 6.2 拾取-放置顺序优化

每对任务需要在两个送料器和两个贴装点之间确定最优移动顺序。共有 4 种排列：

\[
\begin{aligned}
(1)\ &\text{f1 → f2 → a → b} \\
(2)\ &\text{f1 → f2 → b → a} \\
(3)\ &\text{f2 → f1 → a → b} \\
(4)\ &\text{f2 → f1 → b → a}
\end{aligned}
\]

枚举 4 种取最小值，确定每对任务的最优内部顺序。

### 6.3 结果

| 策略 | 总距离 (mm) | 任务数 | 配对 | 单件 | vs 单件 |
|------|------:|:--:|:--:|:--:|:--:|
| 单件取放 (Q=1) | 36,324 | 100 | 0 | 100 | — |
| 预拾取 (Q=2, 同槽) | 23,688 | 60 | 40 | 20 | ↓34.8% |
| **双向取料 (Q=2, 跨槽)** | **22,842** | **50** | **50** | **0** | **↓37.1%** |

双向取料比预拾取再节省 3.57%（846 mm），核心优势在于**消灭了 20 个单件任务**——每个吸嘴往返均完成 2 个贴装，效率最大化。

### 6.4 跨槽配对的物理可行性

跨槽拾取要求相邻两个送料器同时处于吸嘴的 Y 轴行程范围内。SMT 贴片机的吸嘴通常可在 Y 方向小范围微调（±5 mm），而相邻送料器间距仅 10 mm（X 方向），在机械公差允许范围内。此外，现代多头贴片机（如 Fuji NXT 系列）支持独立 Z 轴控制，可同时拾取不同高度位置的元件，为双向取料提供了硬件基础。

## 七、模型局限与改进方向

1. **拾取/贴装时间的忽略**：当空移极度优化后（如 Q=2 预拾取将任务数降至 60），机械动作时间占比上升，后续可将其纳入目标函数。
2. **槽位分配与路径的耦合**：当前采用"先分配后调度"的两阶段方法。端到端的联合优化（槽位分配 + 路径规划同时求解）理论上可进一步改善，但问题变为 NP-Hard 双层优化。
3. **动态元件消耗**：送料器容量和元件补给时间未考虑，实际生产中需引入周期性补料约束。
