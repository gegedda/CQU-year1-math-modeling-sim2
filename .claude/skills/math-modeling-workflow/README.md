# Math Modeling Workflow — Claude Code Skill

AI-assisted 数学建模竞赛完整工作流。从选题分析、任务规划、算法实现、深度分析到论文提交的全流程协作方案。

## 安装

将本目录放入项目的 `.claude/skills/math-modeling-workflow/`，Claude Code 自动发现。

```bash
mkdir -p .claude/skills
git clone git@github.com:gegedda/math-modeling-workflow.git .claude/skills/math-modeling-workflow
```

## 使用

触发词：数模、数学建模、选哪道题、三天计划、论文模板 等。Claude 自动加载。

或直接调用：`/math-modeling-workflow`

## 内容

- 7 阶段完整工作流（赛前准备 → 论文提交）
- 项目结构模板 + 代码规范 + AI 使用红线
- 23 次真实交互的关键经验与教训

## 来源

基于重庆大学 2026 暑期数学建模模拟赛 B 题实战编写。
